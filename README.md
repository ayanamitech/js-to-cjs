Replace .js with .cjs / .mjs recursively

### Install

```bash
$ npm i -g js-to-cjs
```

To use within the project folder

```bash
$ npm i --save-dev js-to-cjs
```

### How to use

```bash
$ js-to-cjs cjs <dir>

# or

$ js-to-cjs mjs <dir>
```
